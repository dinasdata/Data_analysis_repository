<h1>WeatherViz: A weather data visualization tool built with R shiny</h1>
<p>This a simple web interface that is used to plot weather data. The input can be excel or csv file with columns :
RR, Tmoy,Tmin,Tmax,MSLP,RH500, RH700, FF10, FF200, FF700, FF850. These columns correspond to temperature, rainfall, humidity and wind speed parameters.
Then, the app will show the evolution of each parameters per month and per year. It must be to notice that the input data should be a daily data. The app also contains a filter
to plot each parameters under a specific value. Moreover, a specific tab of the app shows a future value off each parameter, using the ARIMA model. A live version of the app
can be used following this link :</p> https://dinasdata.shinyapps.io/weatherviz/
